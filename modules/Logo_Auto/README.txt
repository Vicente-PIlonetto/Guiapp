HOW TO USE

1 - Coloque os arquivos logo1 e logo2 na mesma pasta do script.
	logo1 deve estar em proporção 1:1 (quadrada).
	logo2 deve estar, preferencialmente, em proporção aproximada de 1:4 (retangular horizontal).
2 - Execute o arquivo gerar_logos.bat com duplo clique.
3 - Aguarde a mensagem de conclusão do processo.

RESULTADO

Ao finalizar, serão gerados:
1 - Um arquivo .ZIP contendo todas as logos.
2 - Uma pasta SAIDA com os arquivos individuais:

logofrente.bmp (350x350)
logonfse.jpg (100x100)
logonfe.bmp (530x340)
logopaf.bmp (332x278)
LOGOTIP.BMP (360x90)
logotip.jpg (360x90)

IMPORTANTE

1 - A logo1 deve obrigatoriamente estar no formato 1:1 (quadrada) para evitar distorções nas imagens geradas.

2 - Caso vá copiar o projeto para outro local, copie a pasta completa, pois existem arquivos ocultos necessários para o funcionamento correto do processo.
